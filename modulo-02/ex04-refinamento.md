**Nível 2**
1. **Registrar itens**
   1.1. Repita: leia código/nome do item e valor
   1.2. Pare quando o cliente indicar que não há mais itens
2. **Calcular subtotal**
   2.1. Some os valores de todos os itens registrados
3. **Verificar e aplicar desconto**
   3.1. Pergunte se o cliente tem cartão fidelidade
   3.2. Se sim, desconto ← subtotal × 0.05
   3.3. Se não, desconto ← 0
4. **Mostrar total**
   4.1. total ← subtotal − desconto
   4.2. Exiba o total na tela
